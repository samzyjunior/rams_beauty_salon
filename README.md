# Rams Touch Beauty & Makeup Lounge Website

Website for Rams Touch Beauty & Makeup Lounge, offering professional makeup services in Madina, Ghana.

## About
This website showcases Ruby's makeup services, including bridal makeup, party makeup, and special occasion makeup. It includes sections for booking services, viewing work, and contacting Ruby.

           
- Responsive design for mobile and desktop
- Booking form for services
- Image gallery of makeup work
- About section with Ruby's story
- Contact information

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Flask (Python)
